# FemMonitor

Lightweight application written in C that monitors the status of various FemboyFinancial services.

Best viewed on a wall-mounted 4:3 monitor with a VGA font.

## Dependencies

- libncurses6

- libcurl4

- libxml2.2

- espeak (not as a library, just the program)
